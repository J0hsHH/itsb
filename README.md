## Installing

- Requires python 3.7
 - download [chromedriver](https://chromedriver.chromium.org/downloads), unzip, move to `/usr/local/bin` (mac os / linux)
 - `pip3 install selenium` 
 tips: i use This version of ChromeDriver Chrome version 80
```
cd Downloads
cd itsb-master
python3 bot.py
```
